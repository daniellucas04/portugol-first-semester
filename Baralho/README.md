# Baralho
```md
Criado por: Jones Mendonça de Souza

Silvia é uma mãe dedicada, porém está cansada de arrumar a bagunça que os filhos fazem ao final do dia. Uma das diversões de seus filhos é jogar baralho, porém toda vez Silvia tem que arrumar o baralho e guardar as cartas para os filhos brincarem novamente no dia seguinte. Os meninos além de não ajudar a mãe são também exigentes, pedindo que a mãe organize as cartas de forma empilhada pelos seus valores correspondentes. Silvia pediu sua ajuda parar implementar um algoritmo que dado as cartas embaralhadas organize-o em montes da ordem da primeira carta lida de cima para baixo. Você vai entender vendo o exemplo de entrada e saída abaixo.
```

## Entrada
```md
A entrada é composta de um vetor de caractere k elementos contendo todas as cartas embaralhadas (apenas as 13 cartas primeiras cartas de valores distintos que Silvia encontrou). O modelo do baralho é o Francês de 52 cartas, veja o exemplo no link abaixo.

https://pt.wikipedia.org/wiki/Baralho
```

## Saída
```md
Você deve organizar as cartas em montes de 4 cartas, na ordem da leitura das cartas de cima para baixo.
```
