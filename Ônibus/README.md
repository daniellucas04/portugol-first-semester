# Ônibus
```md
Verinha é uma ótima assitente de alunos do IFSP campus Barretos, ela que auxilia os alunos no transporte da escola ao refeitório. O grande problema que Verinha enfrenta no dia a dia é saber quantos alunos e quantas viagens serão necessárias para o transporte de todos os alunos do ensino médio. Como você é amigo de Verinha faça um programa para ler o número de alunos que estão fila esperando pelo ônibus e exibir na tela a quantidade de alunos por viagens que serão necessários para o transporte. Exemplo: Se tivermos 30 alunos na fila e a capacidade do ônibus for de 7 alunos, teremos quatro viagens com 7 alunos e uma viagem com 2 alunos.
```

## Entrada
```md
A entrada de cada caso de teste é formada por duas linhas. A primeira linha contém um número inteiro N (1 ≤ N ≤ 2000), indicando o número de alunos que estão na fila. A segunda linha contém, um número inteiro que representa a capacidade do ônibus, ou seja, a quantidade de alunos que o ônibus suporta.
```

## Saída
```md
A saída do seu programa deve apresentar N linhas contendo a quantidade de alunos por viagem necessárias para o transporte.
```
```md
    Entrada 1                    Saída 1
    30                           7
    7                            7
                                 7
                                 7
                                 2
```
