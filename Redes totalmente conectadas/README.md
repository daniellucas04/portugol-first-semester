# Redes totalmente conectadas
```md
Em uma empresa, o analista de redes JUCA XAVES deseja montar uma rede de computadores ligando todos os computadores entre si sem usar um concentrador, ou seja, cada computador terá um cabo ligando no outro e assim por diante. Como a empresa possui várias filiais no Brasil e o número de máquinas variam conforme a infraestrutura de cada filial, o analista JUCA XAVES está com dificuldade de estimar número exato de cabos necessários para a montagem da rede. Como JUCA XAVES já se tornou seu amigo durante o curso técnico em informática aqui no IFSP Barretos, ajude-o a resolver este problema criando um programa que receba como entrada o número de computadores e imprima como saída a quantidade de cabos necessários para ligá-los. Por exemplo: Para 2 computadores é necessário apenas 1 cabo; para 3 computadores 3 cabos; para 4 computadores 6 cabos, para 5 computadores 10 cabos, e assim sucessivamente.
```

## Entrada
```md
A entrada é composta de um número inteiro N indicando o número de computadores existentes na filial da empresa.
```

## Saída
```md
A saída deve apresentar apenas o número de cabos necessários para que eles estejam totalmente conectados.
```
```md
    Entrada 1                      Saída 1
    2                              1
```
```md
    Entrada 2                      Saída 2
    3                              3
```
