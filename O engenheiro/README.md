# O engenheiro
```md
A prefeitura da cidade de Olímpia estado de São Paulo anunciou a construção de um aeroporto internacional na cidade. Os moradores ficaram muito felizes com noticia, pois irá gerar mais de 3.000 vagas de emprego para os moradores. Além de beneficiar o turismo local, o projeto de Olímpia é servir como alternativa ao aeroporto de São José do Rio Preto, que está na maior cidade da região, mas que não tem espaço físico para crescer. Atualmente, Rio Preto (como a cidade é conhecida), possui limitações para receber certos tipos de operações, já que sua pista possui apenas 1.640 metros.

O famoso engenheiro e arquiteto Rafael Claassen Damian, foi informado de que o terreno destinado à construção do aeroporto ainda não foi demarcado, mas será retangular. Além disso, o aeroporto deverá ser dividido em quatro zonas, uma delas destinada a uma reserva ambiental e cada uma das outras três receberá os novos prédios hoteleiros. Com muita criatividade, Rafael Claassen Damian decidiu que duas avenidas, perpendiculares entre si, cada uma paralela a dois dos lados do terreno retangular, dividirão o aeroporto nas quatro zonas.

No entanto, a Fundação dos animais silvestres determinou que a área destinada à reserva ambiental era muito pequena, o que obrigou o prefeito a alterar as áreas das quatro zonas. Após receber as novas medidas, Rafael Claassen Damian tentou encontrar um novo retângulo que viabilizasse seu projeto, porém sem sucesso. Cansado de fazer testes, ele pensou que talvez tenha que abandonar sua brilhante ideia. Por isso, ele pediu para você escrever um programa que, dadas as áreas das quatro zonas, determine se ele poderá ou não manter seu projeto (ou seja, se existe um retângulo que possa ser dividido por duas retas perpendiculares, cada uma paralela a dois dos lados do retângulo, tal que as quatro áreas formadas obedeçam às exigências da prefeitura).
```

## Entrada
```md
A entrada consiste quatro inteiros A1, A2, A3, A4, indicando a área de cada uma das zonas.
```

## Saída
```md
Imprima uma única linha contendo um único caractere: ‘S', se Rafael Claassen Damian pode preservar seu projeto e 'N', caso contrário.
```

## Exemplos
```md
Entrada 1                   Saída 1
1                           N  
2                                      
3
4   
```
